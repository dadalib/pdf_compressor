  ```
  pip  install Pillow
  ```

### Installation

1. Get a free API Key at [https://developer.ilovepdf.com/login](https://developer.ilovepdf.com/login)
2. Clone the repo
   ```sh
   git clone https://github.com/github_username/repo_name.git
   ```
3. Install  packages
   ```sh
   npm install
   ```
